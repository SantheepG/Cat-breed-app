A simple iOS app built with SwiftUI, SwiftData, and MVVM architecture that fetches and displays cat breed information from The Cat API.
Features include:

Clean onboarding screen (SwiftData-ready)

Async/await networking with error handling

Decoding and displaying breed details (name, origin, life span, description)

Scrollable list UI with reusable components

MVVM structure for clarity and scalability

A lightweight, well-structured educational project demonstrating modern iOS development practices.
